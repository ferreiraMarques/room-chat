
# Chat Room V1

Chat room for send messages 



## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

#### frotend 
`URL`: backend url

#### server
`FRONT_URL`: frontend url




## Installation

Install chat room with npm


```bash
  cd server
  npm i
  npm run start
```

```bash
  cd frontend
  npm i
  npm run start
```
